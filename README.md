# 🔍 AI Semantic Search Engine

## 🚀 Overview
A full-stack AI-powered search engine that retrieves results based on semantic meaning using transformer models.

## 🧠 Features
- Semantic search using Sentence Transformers (MiniLM)
- FastAPI backend
- Interactive frontend UI
- Real-time results

## 🛠 Tech Stack
- Python
- FastAPI
- Sentence Transformers
- HTML/CSS/JavaScript
